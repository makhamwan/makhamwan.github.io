Web Browsers

A browser is a software application used to locate, retrieve and display content on the World Wide Web, including Web pages, images, video and other files. It is a group of structured codes which together performs a series of tasks to display a web page on the screen. According to the tasks they perform, these codes are made as different components.



  • User Interface The user interface is the space where User interacts with the browser.

  • Browser Engine It works as a bridge between the User interface and the rendering engine. According to inputs from various user interfaces, it queries and manipulates the rendering engine.

  • Rendering Engine The rendering engine, as the name suggests is responsible for rendering the requested web page on the browser screen.

  • Networking Component of the browser which retrieves the URLs using the common internet protocols of HTTP or FTP.

  • JavaScript Interpreter It interprets and executes the javascript code embedded in a website.

  • UI Backend UI backend is used for drawing basic widgets like combo boxes and windows.

    The browser slices the URL into 3 parts



  • http:// The target document can be retrieved from web server, which understands the HTTP protocol. The HTTP protocol is language of communication between browsers and web servers.

  • www.samplelink.com This is the host name of the computer from which the document can be downloaded. 

  • sample.html This is the virtual path of the document in the www.samplelink.com's web server.
